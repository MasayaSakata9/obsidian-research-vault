## Final Verified Summary

**Title:** Active operator learning with predictive uncertainty quantification for partial differential equations

**One-paragraph summary:**
Winovich et al. (arXiv:2503.03178v4, Feb 2026) propose a lightweight predictive uncertainty quantification (UQ) framework for neural operator networks that avoids the computational overhead of ensembles or Bayesian methods. For DeepONet, UQ is integrated by splitting the final branch/trunk layers to produce separate mean (μ) and log-variance (log σ) outputs. For FNO, an external secondary network learns the error distribution of a primary FNO. Both approaches use negative log-likelihood (NLL) loss to calibrate uncertainty estimates to observed model errors. The framework produces unbiased, non-skewed uncertainty estimates that increase appropriately on out-of-distribution data. A precomputed trunk output + sparse placement matrix strategy achieves >5× inference speedup. The UQ estimates are demonstrated to improve data efficiency and accuracy in Bayesian optimization and active learning loops. The paper acknowledges that separating epistemic from aleatoric uncertainty remains an open problem. No code is released.

**Usefulness for the current research:**
**High — for the UQ/calibration component specifically.** The NLL loss + log σ parameterization can be directly applied to the existing 2D FNO baseline for indoor WiFi field prediction. The external UQ pattern (secondary error-learning network) is the most relevant architecture pattern. Uncertainty maps could identify regions (source-near, wall-shadow, far-field) where the model is least confident, guiding targeted data collection. However, validation on 2.4GHz Helmholtz physics is needed — the paper does not test on EM wave propagation.

**Implementation implication:**
1. Split FNO final layer into μ and log σ heads → NLL loss (low cost).
2. Design external UQ secondary network for FNO (medium cost — architecture not specified in paper).
3. Build calibration plots (predicted σ vs actual error) as a new validation metric.
4. Active learning loop using uncertainty for sample selection (medium-high cost, epistemic/aleatoric risk).
